How It Works

Multimodal sensor fusion and behavioral intervention pipeline

1

Sensor Data Collection

ESP32 continuously reads MPU6050 accelerometer/gyroscope data at 50Hz and MQ9 gas sensor at 1Hz. The 6-axis IMU detects hand-to-mouth gestures while the gas sensor identifies volatile organic compounds from cigarette smoke.

2

Edge AI Inference

On-device TinyML model (trained via Edge Impulse) performs spectral analysis on sensor streams. When gesture pattern + smoke signature align, the model classifies the event as "smoking" with confidence score.

3

BLE Notification

SmartQuit Band sends encrypted event notification via Bluetooth Low Energy to BreatheFree app. Only high-confidence events (>85%) are transmitted, preserving battery life.

4

Immediate Intervention

BreatheFree app prompts user to log triggers and mood state. Launches contextual intervention (breathing, games, mindfulness) based on detected emotional state and location.

5

Multi-Stakeholder Insights

Dashboard analytics accessible to user, family members (with consent), and rehab consultants. Temporal heatmaps, trigger analysis, and progress metrics inform personalized support strategies.
